1. Platform Scope & Purpose

FraudShield is a frontend-only web application developed as a proof-of-concept and educational demonstration platform. It is designed to simulate a fraud detection and reporting system. This platform does not process real fraud reports, connect to law enforcement, or transmit data to any external server.

2. Data Storage & Privacy

All data entered into this platform — including login credentials, fraud reports, and profile information — is stored exclusively in your browser's localStorage. This data does not leave your device. No backend server, database, or external API is involved in any data storage or retrieval.

3. No Real Authentication

Login credentials provided in this platform are hardcoded for demonstration purposes only. There is no real password security, encryption, or identity verification. You should never use real personal passwords on this platform.

Users must not submit real personally identifiable information, real fraud case details that require legal action, or any sensitive personal data.
